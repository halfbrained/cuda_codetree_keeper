Plugin for CudaText.
Prevents Code tree fold-state resetting on document changes and app restarts.


Plugin configuration
--------------------
Plugin has a config file, which can be opened in CudaText by:
  "Options / Settings-plugins / Code-Tree Keeper / Config".
  
Available option:
  max_history - max number of documents for which the tree state is kept. Default is 16.
